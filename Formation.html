div, span et section

div

Le <div> est utilisé pour diviser le contenu en sections distinctes et est souvent utilisé pour regrouper d'autres éléments HTML pour les styles ou la manipulation JavaScript.

voir plus

span

Le <span>, quant à lui, est un élément en ligne qui est utilisé pour appliquer un style à une partie de texte ou pour cibler des portions spécifiques du contenu pour le stylage ou le scripting.

voir plus

section

En HTML, la balise <section> est utilisée pour diviser le contenu d'une page web en sections ou zones thématiques. Elle est souvent utilisée pour regrouper des contenus connexes, tels que des articles, des chapitres, des groupes de contenu liés, etc. L'utilisation de <section> aide à structurer et à organiser le contenu de manière logique et sémantique. Chaque section peut avoir un titre défini par une balise de titre comme <h1>, <h2>, etc., pour indiquer le sujet ou le thème de cette section.

En HTML, les tableaux sont des éléments qui permettent d'organiser les données sous forme de lignes et de colonnes. Voici les principaux éléments utilisés pour créer des tableaux :

  • <table> : C'est l'élément de base qui définit le tableau.
  • <tr> : Cet élément représente une ligne dans le tableau.
  • <th> : Utilisé pour créer des cellules d'en-tête (header) dans le tableau. Le contenu de ces cellules est généralement en gras et centré par défaut.
  • <td> : Cet élément crée des cellules de données (data) dans le tableau. Le contenu de ces cellules est généralement aligné à gauche.

La balise <ul> est utilisée pour créer une liste non ordonnée, où chaque élément est marqué par une puce. Elle est souvent utilisée pour afficher des éléments sans ordre spécifique.

La balise <ol> est utilisée pour créer une liste ordonnée, où chaque élément est numéroté. Elle est souvent utilisée pour afficher des éléments dans un ordre spécifique.
